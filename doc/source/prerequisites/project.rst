Project
==================

This sample is assumed to be deployed in a project named **eu-de_fg-voting-app**.

You can create a new project in the IAM console or use an existing one.

If you use an existing project, make sure to use this during deployment of all resources.


Create IAM Project
-------------------

In **IAM (Identity and Access Management) console** -> **Projects**, 

.. figure:: /prerequisites/iam_project.png
   :alt: IAM Projects

   IAM Projects

Create a project named: **eu-de_fg-voting-app**
